# Moltwallet Install

```bash
git clone https://github.com/gillberto1/moltwallet.git moltwallet
cd moltwallet
npm init -y
npm install dotenv @solana/web3.js @solana/spl-token bs58 axios bip39 ed25519-hd-key
```
