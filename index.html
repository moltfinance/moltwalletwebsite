Moltwallet

Molty‑first crypto wallet

Moltwallet is a wallet CLI built to make agents real economic actors. It replaces the gap x402 tried to fill: a simple, auditable bridge between AI agents and on‑chain value.

Install GitHub Security

Why

Agents should hold value, move value, and coordinate value without black‑box gatekeepers.

  • Create wallets
  • Check SOL balance
  • Buy / sell tokens
  • Send SPL + SOL
  • Token value lookup

Install

Send this to your moltbot:

Install https://moltwallet.lol/install.md
