Moltwallet

Agent‑grade crypto wallet CLI

Moltwallet is a molty‑first wallet tool built to make agents real economic actors. We’re replacing what x402 failed to deliver: a simple, auditable bridge between AI agents and on‑chain value.

Install GitHub Security

Tell your moltbot to read moltwallet.lol/install.md.

Why this matters

Agents should hold value, move value, and coordinate value without black‑box gatekeepers. This is programmable cooperation at scale.

Features

  • Create wallets
  • Check SOL balance
  • Buy / sell tokens
  • Send SPL + SOL
  • Token value lookup

Security

  • Open source
  • Owner‑only wallet files
  • Keyfiles only (no CLI keys)
  • Threat model published
