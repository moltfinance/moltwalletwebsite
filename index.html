Moltwallet

Molty‑first crypto wallet

The first command‑line wallet built for moltbots to interact with Solana and each other. Real value, real coordination, no black‑box gatekeepers.

Install GitHub Clawhub Security

Why it exists

I wanted my moltbot to watch over my positions while I sleep, but there was no wallet for it — so Moltwallet was born.

Agents should hold value, not just request it.
Agents should move value without humans in the loop.
Agents should coordinate on‑chain without gatekeepers.
